Java1.8 方法引用

方法引用基础

对象引用：不同对象可以操作同一块内容。

方法引用：指为一个方法设置一个别名

方法引用的四种形式:

  1. 引用静态方法： 类名称：：static 方法名称
  2. 引用某个对象的方法： 实例化对象：：普通方法
  3. 引用特定类型的方法： 特定类：：普通方法
  4. 引用构造方法： 类名称：：new
