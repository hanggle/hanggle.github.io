14:T1180,<h1 id="heading-0">TailwindCSS 实用技巧分享</h1>
<p>TailwindCSS 是一个实用优先的 CSS 框架，它通过提供大量的原子类来帮助开发者快速构建界面。</p>
<h2 id="heading-1">为什么选择 TailwindCSS？</h2>
<h3 id="heading-2">优势</h3>
<ul>
<li><strong>快速开发</strong> - 无需离开 HTML 即可设计界面</li>
<li><strong>一致性</strong> - 预定义的设计系统确保视觉一致性</li>
<li><strong>响应式</strong> - 内置响应式设计支持</li>
<li><strong>可定制</strong> - 高度可配置的设计系统</li>
<li><strong>性能</strong> - 只包含使用的样式</li>
</ul>
<h3 id="heading-3">与传统 CSS 的对比</h3>
<pre><code class="language-html" data-language="html">&#x3C;!-- 传统 CSS -->
&#x3C;div class="card">
  &#x3C;h2 class="card-title">标题&#x3C;/h2>
  &#x3C;p class="card-content">内容&#x3C;/p>
&#x3C;/div>

&#x3C;!-- TailwindCSS -->
&#x3C;div class="bg-white rounded-lg shadow-md p-6">
  &#x3C;h2 class="text-xl font-bold mb-4">标题&#x3C;/h2>
  &#x3C;p class="text-gray-600">内容&#x3C;/p>
&#x3C;/div>
</code></pre>
<h2 id="heading-4">实用技巧</h2>
<h3 id="heading-5">1. 响应式设计</h3>
<pre><code class="language-html" data-language="html">&#x3C;div class="w-full md:w-1/2 lg:w-1/3">
  &#x3C;!-- 移动端全宽，平板半宽，桌面1/3宽 -->
&#x3C;/div>
</code></pre>
<h3 id="heading-6">2. 状态变体</h3>
<pre><code class="language-html" data-language="html">&#x3C;button class="bg-blue-500 hover:bg-blue-600 focus:outline-none focus:ring-2 focus:ring-blue-500 focus:ring-opacity-50">
  点击我
&#x3C;/button>
</code></pre>
<h3 id="heading-7">3. 自定义组件</h3>
<pre><code class="language-css" data-language="css">@layer components {
  .btn-primary {
    @apply bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded;
  }
}
</code></pre>
<h3 id="heading-8">4. 暗色模式</h3>
<pre><code class="language-html" data-language="html">&#x3C;div class="bg-white dark:bg-gray-800 text-black dark:text-white">
  支持暗色模式的内容
&#x3C;/div>
</code></pre>
<h2 id="heading-9">最佳实践</h2>
<h3 id="heading-10">1. 合理使用 @apply</h3>
<pre><code class="language-css" data-language="css">/* 好的做法 - 常用组件 */
.card {
  @apply bg-white rounded-lg shadow-md p-6;
}

/* 避免过度使用 */
.complex-component {
  @apply bg-white p-4 rounded shadow border border-gray-200 text-gray-800 hover:shadow-lg transition-shadow duration-200 ...;
}
</code></pre>
<h3 id="heading-11">2. 配置定制</h3>
<pre><code class="language-javascript" data-language="javascript">// tailwind.config.js
module.exports = {
  theme: {
    extend: {
      colors: {
        brand: {
          50: '#f0f9ff',
          500: '#3b82f6',
          900: '#1e3a8a',
        }
      },
      fontFamily: {
        sans: ['Inter', 'sans-serif'],
      },
    },
  },
}
</code></pre>
<h3 id="heading-12">3. 使用 Prettier 插件</h3>
<p>安装并配置 <code>prettier-plugin-tailwindcss</code> 来自动排序类名：</p>
<pre><code class="language-bash" data-language="bash">npm install -D prettier prettier-plugin-tailwindcss
</code></pre>
